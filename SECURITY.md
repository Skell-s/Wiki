# Políticas de Colaboración

En este texto se presenta información respecto a cómo solicitar
el anexo de artículos y el reporte de errores.

## Reportar Error

Vía Pull Requests.

## Solicitar Anexo de Artículos

Vía Pull Requests, modificando el archivo "articles.js" con el markdown
correspondiente de su artículo.

## Nota

realizar dichas acciones colaborativas.
